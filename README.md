# Quest Ace Dark

A dark-themed quiz application designed for ITI students to practice and master their subjects.

## Features

- **Dark Theme**: Easy on the eyes for long study sessions.
- **Comprehensive Question Bank**: Covers ICTSM, WCS, Engineering Drawing, and Employability Skills.
- **Detailed Explanations**: Every question comes with a clear explanation of the correct answer.
- **Practice Mode**: Test your knowledge topic by topic.
- **Bookmarks**: Save difficult questions for later review.
- **Wrong Answers Tracking**: Focus on areas where you need improvement.

## Tech Stack

- **Frontend**: React, Vite, Tailwind CSS, Shadcn UI
- **Routing**: React Router (HashRouter for GitHub Pages compatibility)
- **State Management**: Zustand
- **Data Fetching**: TanStack Query (React Query)

## Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm or pnpm

### Installation

1. Clone the repository:
   ```sh
   git clone https://github.com/girid9/quest-ace-dark-fixed.git
   ```
2. Install dependencies:
   ```sh
   npm install
   ```
3. Start the development server:
   ```sh
   npm run dev
   ```

## Deployment

The project is configured for automated deployment to GitHub Pages via GitHub Actions. Any changes pushed to the \`main\` branch will be automatically built and deployed.
